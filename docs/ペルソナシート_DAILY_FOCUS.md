## 4. 課題とニーズ

### 抱えている課題・悩み

1. 優先順位が決められず、1日のスタートが遅れる
   - やることが複数あると、どこから着手すべきか迷ってしまい、仕事に入るまで時間がかかる。
   - 現在はNotionやメモアプリにタスクを書き出して整理しているが、結局迷うことが多い。

2. 集中したい時間に通知や別タスクで気が散る
   - 会議、チャット、メール確認が多く、まとまった集中時間を作りづらい。
   - 現在はスマホのタイマーや作業用動画を使って集中しようとしている。

3. 1日を振り返ったときに達成感が薄い
   - 細かい作業で1日が終わり、重要なことが進んだ実感を持ちにくい。
   - そのため、自己肯定感が下がり、翌日のモチベーションにも影響する。

### サービスに期待すること

- 今日最も重要なことをすぐ決められること
  - タスクを増やすのではなく、その日の軸を明確にできることを求めている。
- 集中に入りやすい仕組みがあること
  - タイマーや画面設計によって、迷わず作業に入れる体験を期待している。
- 振り返りを簡単に残せること
  - 長い記録ではなく、短い入力で達成感や改善点を見える化できることが理想である。
